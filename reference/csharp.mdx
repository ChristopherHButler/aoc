C# is a statically typed language. This means that you must declare the type of a variable when you declare it.

### Variables

syntax:

```csharp

```

```csharp

float b = 10.5f;

char d = 'a';
```


### switch

classic switch statement:

```csharp
switch (variable)
{
		case value1:
				// code
				break;
		case value2:
				// code
				break;
		default:
				// code
				break;
}
```

new switch statement:

```csharp
variable switch
{
		value1 => // code,
		value2 => // code,
		_ => // code
};
```

### Functions

In C#, all functions must be defined within a class or struct.

You cannot have standalone functions outside of a class or struct. This is different from some other languages like Python or TypeScript, where you can define functions at the global scope.


## Object Oriented Programming

### classes

A class is a blueprint for creating objects, providing initial values for state (fields) and implementations of behavior (methods).

```csharp
public class ClassName
{
    // Fields
		// Fields: Variables declared within a class to hold data.
    private int field;

    // Properties
		// Properties: Provide a way to read, write, or compute the values of private fields.
    public int Property { get; set; }

    // Constructor
		// Constructor: A special method called when an instance of the class is created.
    public ClassName(int initialValue)
    {
        field = initialValue;
    }

    // Methods
		// Methods: Functions defined within a class to perform actions.
    public void Method()
    {
        Console.WriteLine("Method called");
    }
}
```

### Access Modifiers

Access modifiers control the visibility of classes, fields, properties, and methods.

 - `public`: Accessible from any other code.
 - `private`: Accessible only from within the same class.


### Static Classes and Members

A static class cannot be instantiated and can only contain static members.

```csharp
public static class StaticClass
{
		public static int StaticField = 10;

		public static void StaticMethod()
		{
				Console.WriteLine("Static method called");
		}
}
```




## LINQ

LINQ (Language Integrated Query) allow you to query data in C#.

You can make use of extension methods syntax or query syntax.

### Extension Methods Syntax

```csharp
var books = new BookRepository().GetBooks();

var cheapBooks = books
                    .Where(b => b.Price < 10) // filter
                    .OrderBy(b => b.Title)    // sort
                    .Select(b => b.Title);    // fields to populate
```
